# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-17

### Added
- Initial `currency-localize` package release.
- Immutable `Money` value object with arithmetic and allocation.
- Locale-aware parse and format APIs.
- Country-to-currency resolution.
- Deterministic serialization and trace metadata support.
- CI, release, npm publish automation, and branch protection.
